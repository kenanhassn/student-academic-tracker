# Student Academic Tracker

This is a simple console-based C++ project designed to manage student academic records.

## Features
- Add student information
- Enter grades for multiple subjects
- Calculate and display student average
- Search for a student by ID

## Technologies Used
- C++
- Functions
- Arrays
- Structs

## Purpose
This project was created to practice programming fundamentals and structured programming concepts in C++.

## How to Run
1. Compile the program using any C++ compiler.
2. Run the executable in the console.
3. Follow the on-screen menu.
